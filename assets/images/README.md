# assets/images/

## Portrait Photo

Place your portrait photo here:
- **Filename:** `portrait.jpg` (or `portrait.png`)
- **Recommended size:** Minimum 600×600px, square crop preferred
- **Format:** JPG or PNG

Then follow the instructions in the main README.md to enable the photo
in the portfolio (search for "PORTRAIT IMAGE" in index.html).

---

You can also add project screenshots here and reference them in the
project cards in index.html.
